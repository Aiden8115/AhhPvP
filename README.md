# AhhPvP
A Minecraft Mod by Ahh Studio

## Introduction
A Mod helping PvP. Items: 
- Water Bottle + Glass(Brewing recipe): Invisibility Potion 30s
- Throwable Fireball(createFire, breakBlocks)
- Rescue Platform(CD 20s, blocks placed by it don't drop)
- Auto TNT Ignite(That's the custom one)
- 1.8 Fishing Rod
- Bridge Egg
- 1.8 Sword
- Bed Bug
- So on...
